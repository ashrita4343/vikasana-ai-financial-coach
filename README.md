# Vikasana AI Financial Coach 💰🤖

Vikasana AI Financial Coach is an AI-powered React and TypeScript web application designed to provide intelligent financial guidance through interactive dashboards and coaching features.

## ✨ Features

- 🤖 AI-powered financial coaching
- 📊 Interactive financial dashboard
- 💡 Smart financial insights
- 📈 Financial guidance and recommendations
- 🎨 Modern React-based user interface

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Gemini AI integration

## 📂 Project Structure

```
vikasana-ai-financial-coach/
│
├── App.tsx
├── components/
├── hooks/
├── services/
├── index.tsx
├── package.json
└── vite.config.ts
```

## 🚀 Run Locally

### Install dependencies

```bash
npm install
```

### Configure API Key

Create a `.env.local` file and add your Gemini API key:

```
GEMINI_API_KEY=your_api_key_here
```

### Start the application

```bash
npm run dev
```

Open the localhost URL shown in the terminal.

## ⚠️ Note

Keep API keys private and do not upload sensitive information to GitHub.
